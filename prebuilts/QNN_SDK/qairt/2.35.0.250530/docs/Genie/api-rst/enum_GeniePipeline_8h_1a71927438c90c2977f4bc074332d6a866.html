enum GeniePipeline_Priority_t¶

An enum which defines the pipeline priority.

Values:

enumerator GENIE_PIPELINE_PRIORITY_LOW = 0¶

GENIE_PIPELINE_PRIORITY_LOW is always available for use.

enumerator GENIE_PIPELINE_PRIORITY_NORMAL = 100¶

GENIE_PIPELINE_PRIORITY_NORMAL is always available for use. This is the default.

enumerator GENIE_PIPELINE_PRIORITY_NORMAL_HIGH = 150¶

GENIE_PIPELINE_PRIORITY_NORMAL_HIGH usage may be restricted and would silently be treated as GENIE_PIPELINE_PRIORITY_NORMAL

enumerator GENIE_PIPELINE_PRIORITY_HIGH = 200¶

GENIE_PIPELINE_PRIORITY_HIGH usage may be restricted and would silently be treated as GENIE_PIPELINE_PRIORITY_NORMAL
